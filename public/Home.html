Home

Welcome to the Group Management System

The goal of this website is to allow users to create and adjust their own personal project groups, develop a checklist of items the group is required to complete which can only be added or removed by the group leader, and to allow members of the group to check off completed tasks.

Description

Home

Contains the website description and the page descriptions.

Projects

Contains the current user's list of projects and groups.

Note: User must be logged on in order to view them.

About Me

Contains the details about the developer as well as the references used to make this website.

Sign up

Allows new users to sign up and use the website.

Log in

Allows users to access their projects and groups in the Projects tab. New users must sign up first before they can log in.
